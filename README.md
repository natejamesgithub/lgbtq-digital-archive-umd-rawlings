# 🌈 LGBTQ+ DC/DMV Digital Archive

A digital archive dedicated to preserving and organizing LGBTQ+ history in Washington, D.C., Maryland, and Virginia.

This project aims to make regional queer history easier to explore, research, and remember.

---

## What This Archive Is

The LGBTQ+ DC/DMV Digital Archive is a structured collection of:

- Organizations  
- Events  
- Publications  
- Landmarks  
- Notable people  

It serves as a growing historical resource for students, researchers, and the community.

---

## Purpose

- Preserve local LGBTQ+ history  
- Centralize scattered historical information  
- Create an accessible and organized digital resource  
- Build a foundation for future expansion  

---

## Current Features

- Structured archive pages by category  
- Clean and responsive layout  
- Organized folder structure (HTML, CSS, JS separated)  
- Internal navigation between sections  
- Scalable foundation for future dynamic features
